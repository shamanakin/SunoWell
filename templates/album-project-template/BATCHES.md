# Batches

This file logs each batch you run so you can iterate without amnesia.

Append new batches at the top.

---

## Batch template

```text
## [YYYY-MM-DD] SW-BATCH-<id>
- Objective: album | singles
- Tracks generated: <count>
- Knob changed (one): <what changed>
- Keepers: <TrackIDs>
- Misses: <TrackIDs>
- Notes: <2 to 5 bullets>
- Suno links/IDs: <paste list or link list>
```
